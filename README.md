# Invoice-System

https://lop-invoice-system.herokuapp.com/

## Introduction
Invoice System project made using NodeJS,Express,Handlebars and MongoDB

## Installation

After installing Node.js:

Import files...

> npm i

Then run:

> nodemon index.js
